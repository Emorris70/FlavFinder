# User Stories
M = MVP </br>
NM = Non-MVP

---
## MVP User Stories

---
### Personalized Profile & Security (M)

###  Sign up (M) 

As a first-time user, I want to create a secure account with my email and unique password.

### Login (M)

As a registered user, I want to quickly sign back into my account using my credentials.

### Password Reset (M)

As a registered user or admin user, I want to ensure I can reset my password if I forget.

### Location Intelligence (M)

As a hungry user on the move, I want to grant permission for FlavFinder to access my current
GPS coordinates,</br> so that I can see the closest food options immediately without having to manually 
type in my address.

### Favorites (M)

As I discover new restaurants I would like to save it to a list, so I come back to it later.

### Remove Item from favorites (M)

If I accidentally favorite a restaurant or decide I want to clear my favorites,
I should have the option to easily remove that restaurant from my list.

## Filtered Search / General Search (M)

### Search (M)

As a hungry user with a specific taste in mind,
I want to type a cuisine (like "Sushi" or "Burgers") into the search bar while using my current location, 
and get matching results near me.

### Location Change (M)

As a user who is either at home or planning a trip,
I want to toggle between my current GPS location and a manually entered city or zip code,</br>
So that I can explore dining options in my immediate vicinity or scout a completely different area.

### Filtered Search (M)

As a hungry user with specific tastes and a set budget,
I want to filter my results by cuisine type and price point ($, $$, $$$),</br>
So that I only see restaurants that I actually want to eat at and can afford.

---

## Non-MVP User Stories

### Search Intelligence (NM)

As a specific craver or traveler,
I want to enter a custom keyword or a specific location into a search bar,
So that I can find exactly what I’m looking for regardless of my current GPS position

### AI Assistance (NM)

As a designated group planner, I want to input multiple conflicting cuisine cravings into an AI-powered assistant, 
So that I can get a curated "best of all worlds" recommendation that prevents a group argument.

### Random Generator (NM)

As a culinary explorer who is bored with my usual routine, I want to trigger a “Surprise Me” feature based on my past likes but with a “wildcard” twist, 
So that I can discover a hidden gem I would have never searched for myself.

### Leave A Review (NM)

As I visit the restaurants that peak my interest I would like to leave a review for others.
